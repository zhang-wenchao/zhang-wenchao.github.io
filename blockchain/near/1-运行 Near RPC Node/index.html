Docker 运行 Near RPC Node

发表于 2022-03-09 分类于 Blockchain ， NEAR

工作需要自己搭建一个 Node 比较方便，仅做记录。

前置条件

  • Debian
  • Docker
  • Docker Compose
  • AWS CLI

创建

编辑 docker-compose.yml

在同目录下创建 ./data/near/testnet 文件夹。

其他

启动

docker-compose up -d

查看日志等

docker exec nearup nearup logs -f
