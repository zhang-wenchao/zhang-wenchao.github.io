frp 内网穿透极简使用指南

发表于 2022-03-07 分类于 Devops

frp 算是使用过的最简单的一个内网穿透服务。

下载

下载适合自己平台的压缩包，服务器和本地都需要。

https://github.com/fatedier/frp/releases

解压缩

tar -zxvf frp_*.tar.gz

服务器配置

编辑：vim frps.ini

启动：./frps -c ./frps.ini

客户端配置

编辑：vim frpc.ini

启动：./frpc -c ./frpc.ini

总结

经过如此简单的配置，成功代理本地无公网 IP 机器上的 Web HTTPS 服务。代理的是 TCP 其他服务也没有问题，官网有更详细的文档。https://gofrp.org/docs/，开始的第一步是成功运行服务，更高级的使用可以慢慢学习添加。
